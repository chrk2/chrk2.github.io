Политика конфиденциальности

1. Общие положения

Настоящая политика обработки персональных данных составлена в соответствии с Законом Республики Казахстан «О персональных данных и их защите» и определяет порядок обработки персональных данных и меры по обеспечению их безопасности Товариществом с ограниченной ответственностью «Vizit» (далее — «Оператор»).

Оператор осуществляет обработку персональных данных пользователей сайта Vizit.kz и принимает необходимые меры для их защиты.

2. Какие данные мы собираем

2.1. При оформлении заказа пользователь может предоставить следующие данные:

— имя;
— контактный телефон;
— адрес доставки (при необходимости).

2.2. Сайт автоматически может собирать техническую информацию, включая:

— IP-адрес;
— тип браузера;
— данные о времени посещения сайта;
— информацию о действиях на сайте.

3. Цели обработки персональных данных

3.1. Обработка и выполнение заказов пользователей.

3.2. Связь с пользователем для уточнения деталей заказа.

3.3. Организация доставки товара.

3.4. Улучшение качества работы сайта и обслуживания клиентов.

4. Передача данных третьим лицам

4.1. Персональные данные пользователя могут передаваться третьим лицам исключительно в целях выполнения заказа (например, службам доставки).

4.2. Оператор не продаёт и не передаёт персональные данные пользователей третьим лицам для маркетинговых целей.

5. Согласие на обработку персональных данных

5.1. Отправляя данные через формы на сайте Vizit.kz, пользователь выражает согласие на обработку своих персональных данных в соответствии с настоящей политикой конфиденциальности.

6. Защита персональных данных

6.1. Оператор принимает необходимые организационные и технические меры для защиты персональных данных пользователей от неправомерного доступа, изменения, раскрытия или уничтожения.

6.2. Доступ к персональным данным имеют только сотрудники, непосредственно участвующие в обработке заказов.

7. Права пользователя

7.1. Пользователь имеет право запросить информацию о своих персональных данных, их исправление или удаление.

7.2. Для реализации своих прав пользователь может направить запрос на электронную почту: info@vizit.kz.
